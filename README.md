String art to G-code generator
==============
A Python-based graphical user interface (GUI) tool for converting images into circular string-art-inspired fiber deposition paths and generating corresponding G-code files for Melt Electrowriting (MEW) fabrication.

The software transforms grayscale images into line-based patterns defined by connections between nodes distributed along a circular boundary. The resulting node sequence can be directly exported as a printing path for MEW systems.

**Features**
---------
- Import image
- Adjust contrast
- Generate pattern
- Export G-code
- Real-time preview of the generated string-art pattern
- Export node connection sequences
- Automatic generation of G-code for MEW printing
- Customizable parameters, including:
  - node number
  - line number
  - circle diameter
  - printing feed rate

 **Workflow**
---------

- Import a target image

- Adjust image contrast if necessary

- Define generation parameters

- Generate the circular string-art pattern

- Preview the resulting fiber path

- Export the node sequence and MEW G-code file

**Requirements**
---------
The software is provided as a standalone executable (.exe).
No installation of Python or additional libraries is required.
Please click 'String art to G-code.exe' file and click download raw file button to download the software from github.
